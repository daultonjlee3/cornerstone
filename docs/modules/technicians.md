## Technicians

The Technicians screen lists the people who do maintenance work so you can manage who is available and how they are used.

### What this screen is for

The Technicians screen stores information about each technician.  
Each row is a person with details like name, company, and contact info.

### Who typically uses it

Maintenance supervisors  
Operations managers  
Dispatchers  
Admin staff  

### Key things you can do

• View all technicians in your tenant  
• See which company each technician belongs to  
• Open a technician to see more details  
• Use technician information when assigning work and planning schedules  

### Create a new technician

1. From the left sidebar, go to **Technicians**.  
2. Make sure at least one company exists; you cannot add technicians without a company.  
3. Click the **New Technician** or **Create** button if it is available.  
4. Enter the technician’s name and contact details.  
5. Select the correct company and any other key fields.  
6. Click **Save**.

### Layout overview

**Header**  
Shows the page title and may include create actions.

**Technician table**  
Lists technicians with columns such as name, company, and status.  
Click a row to open the technician’s details.

**Technician detail**  
Shows more context that helps you assign work and manage access.

### Typical workflow

Technician added in Technicians  
→ Appears as an option in Work Orders and Dispatch  
→ Assigned to work orders and schedules  
→ Performance shows up in reports and dashboards  

### Tips

• Keep names and contact details correct so communication is easy.  
• Make sure technicians are tied to the right companies, especially in multi-company tenants.  
• Regularly review the list to keep only active technicians enabled.
