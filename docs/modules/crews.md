## Crews

The Crews screen groups technicians into teams so you can schedule them together.

### What this screen is for

The Crews screen lets you define teams of technicians who often work as a group.  
Each row is a crew with a name and a set of members.

### Who typically uses it

Maintenance supervisors  
Dispatchers  
Operations managers  

### Key things you can do

• View a list of crews for your tenant  
• Open a crew to see which technicians belong to it  
• Use crews when assigning or scheduling work (where supported)  

### Create a new crew

1. From the left sidebar, go to **Crews**.  
2. Click the **New Crew** or **Create** button if it is available.  
3. Enter a clear crew name (for example, “Day Shift Team” or “Electrical Crew”).  
4. Add the technicians who should be members of this crew.  
5. Click **Save**.

### Layout overview

**Header**  
Shows the page title and may include a create action.

**Crew list**  
Each row shows a crew name and basic information.  
Click a row to open crew details.

**Crew detail**  
Shows which technicians are part of the crew.

### Typical workflow

Crew created in Crews  
→ Appears as an option in Dispatch or Work Orders  
→ Work orders assigned to the crew instead of a single person  
→ Crew completes work as a team  

### Tips

• Create crews that match how your team naturally works day-to-day.  
• Keep membership up to date when staff change teams.  
• Use clear crew names that supervisors and dispatchers recognize instantly.
