## Vendors

The Vendors screen lists the companies that supply parts, materials, or services to your organization.

### What this screen is for

The Vendors screen helps you track who you buy from.  
Each row is a vendor with information such as name and contact details.

### Who typically uses it

Procurement and purchasing staff  
Inventory managers  
Accounts payable  
Operations managers  

### Key things you can do

• View a list of vendors  
• Search for a vendor by name  
• Open a vendor to see more details  
• Use vendor information when creating products and purchase orders  

### Create a new vendor

1. From the left sidebar, go to **Vendors**.  
2. Click the **New Vendor** or **Create** button if it is available.  
3. Enter the vendor’s name and any key contact information.  
4. Add notes that will help your team know when to use this vendor.  
5. Click **Save**.

### Layout overview

**Header**  
Shows the page title and may include create and search controls.

**Vendor table**  
Lists vendors with basic columns such as name.  
Click a row to open vendor details.

**Vendor detail**  
Shows more context and, where available, connections to products and purchase orders.

### Typical workflow

Vendor created in Vendors  
→ Products linked to that vendor  
→ Purchase orders created using those products  
→ Inventory updated when orders are received  

### Tips

• Use names that clearly match how people refer to each vendor.  
• Keep contact details current so ordering and follow-up are easy.  
• Avoid duplicate vendor records for the same company.
