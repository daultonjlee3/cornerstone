## Units

The Units screen lists individual spaces inside buildings, such as suites, rooms, or apartments.

### What this screen is for

The Units screen is the most detailed level of your location structure.  
Each row is a unit that belongs to a building and property, and can hold assets or work.

### Who typically uses it

Operations managers  
Maintenance supervisors  
Technicians  
Asset managers  

### Key things you can do

• View all units across your properties and buildings  
• See which property and building each unit belongs to  
• Use units when placing assets or work orders  

### Create a new unit

1. From the left sidebar, go to **Units**.  
2. Click the **New Unit** or **Create** button if it is available.  
3. Enter a unit name or number that matches how the space is labeled on-site.  
4. Select the building (and property) it belongs to.  
5. Click **Save**.

### Layout overview

**Header**  
Shows the page title and any create or filter options.

**Unit list**  
Each row is a unit, usually showing the unit name, building, and property.  
Use this information when filtering assets and work orders.

### Typical workflow

Property created  
→ Buildings created under the property  
→ Units created under each building  
→ Assets and work orders linked to the correct unit  

### Tips

• Match unit names to what appears on doors, maps, or leasing systems.  
• Keep units updated when spaces are added, merged, or renumbered.  
• Always choose the correct unit when linking assets and work so history stays accurate.
