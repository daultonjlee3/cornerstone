## Inventory

The Inventory screen shows the parts and materials you keep on hand so you can support maintenance work.

### What this screen is for

The Inventory screen tracks items such as filters, belts, fasteners, and other supplies.  
Each row is an item with information like name and any linked product or vendor.

### Who typically uses it

Inventory managers  
Maintenance supervisors  
Purchasing or procurement staff  
Technicians (for reference)  

### Key things you can do

• View a list of inventory items  
• Search for a part by name or code  
• Open an item to see more details  
• Use inventory information when planning work and purchases  

### Create a new inventory item

1. From the left sidebar, go to **Inventory**.  
2. Click the **New Item** or **Create** button if it is available.  
3. Enter a clear item name that matches how your team talks about the part.  
4. Link the item to a product or vendor if that option is shown.  
5. Add any helpful notes such as where it is stored.  
6. Click **Save**.

### Layout overview

**Header**  
Shows the page title and may include a create button and search box.

**Item table**  
Lists items with key information such as name and related product or vendor.  
Click a row to open the item detail.

**Item detail**  
Shows more context that helps you order or use the part correctly.

### Typical workflow

Item added to Inventory  
→ Used on work orders as needed  
→ Restocked through Products and Purchase Orders  
→ Inventory list stays up to date for planning  

### Tips

• Use simple, standard names so technicians can find parts quickly.  
• Keep item records updated when suppliers or products change.  
• Review your list regularly to remove items you no longer use.
