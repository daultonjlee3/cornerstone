## Buildings

The Buildings screen lists the buildings within each property so you can organize work one level below the site.

### What this screen is for

The Buildings screen is the middle layer of your location structure.  
Each row is a building that belongs to a property and can contain units, assets, and work.

### Who typically uses it

Operations managers  
Maintenance supervisors  
Asset managers  

### Key things you can do

• View all buildings for your tenant  
• See which property each building belongs to  
• Use buildings as filters when viewing assets and work orders  

### Create a new building

1. From the left sidebar, go to **Buildings**.  
2. Click the **New Building** or **Create** button if it is available.  
3. Enter a building name (for example, “Main Warehouse” or “Building A”).  
4. Select the property this building belongs to.  
5. Click **Save**.

### Layout overview

**Header**  
Shows the page title and any create or filter controls.

**Building list**  
Each row is a building with its name and related property.  
Use this information when filtering in other modules.

### Typical workflow

Property created in Properties  
→ Buildings created under that property  
→ Units, assets, and work orders linked to a building  
→ Filters use property and building together to narrow results  

### Tips

• Use names that are easy to recognize on-site.  
• Keep the list current when buildings are added, renamed, or taken out of service.  
• Make sure each building is linked to the correct property.
