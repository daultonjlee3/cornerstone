## Preventive Maintenance

The Preventive Maintenance screen lists maintenance plans that create recurring work before equipment fails.

### What this screen is for

The Preventive Maintenance screen helps you plan repeating maintenance tasks for important assets.  
Each row is a plan that can generate work orders on a schedule.

### Who typically uses it

Maintenance supervisors  
Operations managers  
Planners and schedulers  

### Key things you can do

• View existing preventive maintenance plans  
• Open a plan to see its schedule and linked assets  
• See when work from a plan is next due  

### Create a new maintenance plan

1. From the left sidebar, go to **Preventive Maintenance**.  
2. Click the **New Plan** or **Create Plan** button if it is available.  
3. Enter a clear plan name (for example, “Quarterly Filter Change”).  
4. Select the assets or locations this plan should cover.  
5. Choose how often the work should occur (such as monthly or quarterly).  
6. Click **Save**.

### Layout overview

**Header**  
Shows the page title and any create or filter controls.

**Plan list**  
Each row shows a maintenance plan with basic information such as name and frequency.  
Click a row to open plan details.

**Plan detail**  
Shows which assets are covered and how the work is scheduled.

### Typical workflow

Preventive maintenance plan created  
→ System generates work orders based on the schedule  
→ Technicians complete the scheduled work  
→ Work history builds up on each asset over time  

### Tips

• Focus plans on critical assets where failure would be costly.  
• Use clear plan names so it is obvious what each plan does.  
• Review plans a few times per year to make sure frequencies still make sense.  

## Preventive Maintenance

### What this module is for

The Preventive Maintenance module helps you plan and track recurring maintenance work before equipment fails.  
It turns ongoing maintenance plans into scheduled work orders over time.

### Who uses this module

Maintenance supervisors  
Operations managers  
Planners and schedulers

### When you would use this module

- Setting up repeating maintenance for key assets  
- Reviewing upcoming planned work  
- Checking which plans are active for a company or asset  
- Moving from plans into individual work orders

### Key things you can do here

- View preventive maintenance plans at `/preventive-maintenance`  
- Open a specific plan at `/preventive-maintenance/[id]`  
- See when work is next due for each plan  
- Connect plans to assets and companies

### Main screen overview

When you open **Preventive Maintenance** from the sidebar you see a list of maintenance plans.  
The table shows basic information such as name, frequency, and next due timing where available.  
You can open a plan to see more details, including how it links to assets and how future work is scheduled.

### Step-by-step guide

#### Review preventive maintenance plans

1. Click **Preventive Maintenance** in the left sidebar.  
2. Review the list of plans to see what is set up.  
3. Click on a plan to open its detail page.  
4. Review which assets and schedule rules the plan uses.

#### Check what is coming due

1. From the list of plans, look for any fields that show next due dates or statuses.  
2. Use this information together with **Work Orders** and **Dispatch** to plan upcoming work.

### Common workflows

Preventive maintenance plan created  
↓  
System generates work orders on a schedule  
↓  
Technicians complete work  
↓  
History builds against assets and reports

### Tips and best practices

- Focus preventive maintenance on critical assets that would be costly to fail.  
- Keep plan names clear so they are easy to recognize in lists and reports.  
- Review plans regularly to make sure frequencies and scopes still make sense.
