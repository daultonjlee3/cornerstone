## Work Queue

The Work Queue screen shows each technician’s list of upcoming jobs in a simple, technician-focused view.

### What this screen is for

The Work Queue helps you and your technicians see what is assigned to them in order.  
Each row is a scheduled or assigned work order for a technician.

### Who typically uses it

Technicians  
Maintenance supervisors  
Dispatchers  

### Key things you can do

• View work orders assigned to technicians  
• See basic details like title, status, and schedule  
• Open a work order for full details and updates  

### Create a new assignment

1. Assign work from **Work Orders** or **Dispatch** (see those guides for details).  
2. Once a work order is assigned to a technician, it appears in the technician’s Work Queue.  
3. Technicians can then open the job from the Work Queue to work on it.  

### Layout overview

**Header**  
Shows that you are in the Work Queue view.

**Queue list**  
Displays upcoming work orders for one or more technicians.  
Each row shows a short summary and the current status.  
Click a row to open the full work order.

### Typical workflow

Supervisor assigns work in Dispatch or Work Orders  
→ Assigned work appears in each technician’s Work Queue  
→ Technician opens a job from the queue and completes it  
→ Status is updated and the job drops off the active queue  

### Tips

• Use Work Queue in daily technician briefings to review each person’s day.  
• Encourage technicians to open jobs from the queue and update status as they go.  
• Keep assignments manageable so the queue is not overwhelming.
