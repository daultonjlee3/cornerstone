## Onboarding

The Onboarding screens guide new tenants through the basic setup steps so they can start using Cornerstone.

### What this screen is for

Onboarding helps you create the first companies and basic records your tenant needs.  
It shows simple steps so you do not miss anything important during initial setup.

### Who typically uses it

New tenant admins  
Implementation and onboarding staff  
Platform super admins helping customers  

### Key things you can do

• Complete the first setup steps when your tenant is new  
• Use the Onboarding Wizard to follow a clear sequence of tasks  
• Move from setup into everyday modules like Assets and Work Orders  

### Start onboarding

1. Sign in as a new tenant admin.  
2. If you are missing setup, you may be sent to `/onboarding` automatically.  
3. Or, from the left sidebar, open **Onboarding Wizard** in the **Admin** section.  
4. Follow the on-screen steps to create required records such as companies.  
5. Continue until the wizard shows that setup is complete.

### Layout overview

**Onboarding page (`/onboarding`)**  
Explains what is needed to finish setup and may link directly into the wizard or setup steps.

**Onboarding Wizard (`/onboarding-wizard`)**  
Shows a series of simple screens, each focused on one setup task, such as creating a company.

### Typical workflow

New tenant created  
→ Tenant admin signs in and is guided to Onboarding or the Onboarding Wizard  
→ Admin completes required setup steps  
→ Admin begins using core modules like Operations Center, Work Orders, and Assets  

### Tips

• Use the wizard to keep track of what is left to do; avoid skipping around too much.  
• Once onboarding is complete, train users starting from the Operations Center and key modules.  
• Super admins can use the wizard when assisting customers with their first setup.
