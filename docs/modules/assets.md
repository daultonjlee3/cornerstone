## Assets

The Assets screen lists the equipment and physical items your organization maintains so you can see what you have and where it is.

### What this screen is for

The Assets screen is where you keep track of important equipment.  
Each row represents one asset with information like name, company, and location.

### Who typically uses it

Operations managers  
Maintenance supervisors  
Technicians (for reference)  

### Key things you can do

• View all assets in your tenant  
• Filter assets by company or location  
• Open an asset to see more details  
• Use asset information when creating or reviewing work orders  

### Create a new asset

1. From the left sidebar, go to **Assets**.  
2. Click the **New Asset** or **Create Asset** button if it is available in the header.  
3. Enter a clear asset name that technicians will recognize.  
4. Choose the correct company and location (property, building, and unit if known).  
5. Fill in any extra details such as type or notes.  
6. Click **Save** or **Create**.

### Layout overview

**Header**  
Shows the page title and may include a create button and filters.

**Asset table**  
Lists assets with key columns such as name, company, and location.  
Click a row to open the asset detail view.

**Asset detail**  
Shows more information about the asset and links to related work (where available).

### Typical workflow

Asset created in Assets  
→ Work orders linked to that asset  
→ Technicians complete work  
→ History can be reviewed later through Assets and Reports  

### Tips

• Use simple, consistent names so assets are easy to search.  
• Always link work orders to the right asset when possible.  
• Keep location fields up to date if equipment moves.
